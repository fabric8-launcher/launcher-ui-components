Welcome to the Launchpad

You are moments away from continuous delivery of a new application, built and deployed on the OpenShift cloud platform. Here we'll help you:

  • Choose your Mission, an initial goal like "open up an HTTP endpoint" or "interact with a database".
  • Select a Runtime, one of Eclipse Vert.x for asynchronous and reactive development, WildFly Swarm for Microservices with Java EE, or Spring Boot for opinionated, standalone Spring applications.
  • Access your Booster, a fully-functioning starter application which will be live and available in minutes.

Along the way we'll (with your permission!) push the Booster into your GitHub namespace, set up a build pipeline for continuous delivery, and ensure it's triggered to run on each push to your git remote.

Let's get you into orbit.

For more information, see: Release Notes and Supported Runtime Versions.
