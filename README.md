Reusable UI components for the Launcher UI based on React.js.

If this is the first time you are starting the UI, you need to run

```bash
$ yarn install
```

## Development

### Start Storybook dev server
```bash
$ yarn comp:storybook
```

### Build the library
```bash
$ yarn build
```


## Test

```bash
$ yarn test
```

## Build for production

```bash
$ yarn build
```
